# Time-series profiles of consumption and production energy data

This data set is extracted from
[`flextools::energy_profiles`](https://resourcefully-dev.github.io/flextools/reference/energy_profiles.html)

## Usage

``` r
dtf
```

## Format

A tibble

- solar:

  Solar PV production (in kW)

- building:

  Consumption profile of a building (in kW)
