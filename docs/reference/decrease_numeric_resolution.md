# Decrease resolution of a numeric vector

Decrease resolution of a numeric vector

## Usage

``` r
decrease_numeric_resolution(y, n, method = c("average", "first", "sum"))
```

## Arguments

- y:

  original numeric vector

- n:

  integer, number of intra-values (counting the original value as the
  first one)

- method:

  character, being `average`, `first` or `sum` as valid options

## Value

numeric vector
