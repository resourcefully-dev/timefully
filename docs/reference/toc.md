# Time difference end function

Use this function together with
[`tic()`](https://resourcefully-dev.github.io/timefully/reference/tic.md)
to control time spent by functions

## Usage

``` r
toc(units = "secs", digits = 2)
```

## Arguments

- units:

  character, one of "auto", "secs", "mins", "hours", "days" and "weeks"

- digits:

  integer, number of decimals

## Value

numeric
